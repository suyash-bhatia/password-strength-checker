# password-strength-checker
A Python program that checks the strength of a password based on multiple conditions and provides feedback for improvement.
## Features
- Checks password length
- Detects uppercase and lowercase letters
- Checks for digits
- Checks for special characters
- Provides feedback for improvement

## How it works
The program evaluates the password based on 5 conditions and assigns a score. Based on the score, the password is classified as:
- Weak
- Medium
- Strong

## How to run
1. Run the Python file
2. Enter your password
3. View strength and suggestions

## Tech Used
- Python
